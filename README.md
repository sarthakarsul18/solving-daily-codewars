## 📌 About This Repository

This repository is a **daily log of my Codewars journey** — every kata I solve gets pushed here, organized by difficulty (kyu), so anyone (including future me 👋) can track how the problem-solving skill has evolved over time.

This is **not just a solution dump**. The goal is to build, in public:

- 🧠 Stronger problem-solving & logical thinking
- 🐍 Deeper Python fluency
- 📊 Solid grip on Data Structures & Algorithms
- ✍️ Cleaner, more readable code
- 🔁 A real, visible daily-practice habit
